## **Controls**

### QWERTY

Command             | Action
-------             | ------
W                   | Move forward
A                   | Move left
S                   | Move backward
D                   | Move Right
Left Shift          | Sprint
Left Ctrl           | Dodge
Space bar           | Jump
Esc / P             | Pause / Unpause the game
Move Mouse          | Move the camera around the player
Mouse Right Click   | Shoot
